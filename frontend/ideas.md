## 아이디어 1: Cyberpunk Industrial (사이버펑크 인더스트리얼)

**Design Movement**: 사이버펑크 + 인더스트리얼 브루탈리즘의 융합. 블레이드러너와 같은 미래도시 미학에서 영감을 받되, 산업적 질감과 원시적 구조물의 강렬함을 결합.

**Core Principles**:
1. 어둠 속의 빛 — 깊은 블랙 배경 위에 네온 라인이 정보를 안내
2. 구조적 그리드 파괴 — 대각선 컷, 비대칭 레이아웃으로 긴장감 유지
3. 촉각적 텍스처 — 콘크리트, 금속, 노이즈 그레인으로 디지털 촉감 부여
4. 기능적 미니멀리즘 — 장식 배제, 모든 요소가 목적을 가짐

**Color Philosophy**: 순수 블랙(#0A0A0A)을 캔버스로, 일렉트릭 바이올렛(#A855F7)과 핫 마젠타(#EC4899)가 네온 사인처럼 공간을 가르는 역할. 쿨 그레이(#374151)가 구조적 뼈대를 형성하고, 사이안(#06B6D4)이 보조 악센트로 기술적 느낌을 강화.

**Layout Paradigm**: 풀블리드 히어로 + 대각선 섹션 디바이더 + 오프셋 그리드. 각 섹션이 마치 거대한 디지털 간판처럼 독립적으로 존재하면서도 스크롤로 연결되는 시네마틱 구조.

**Signature Elements**:
1. 글리치 텍스트 효과 — 제목에 RGB 분리와 깜빡임 적용
2. 네온 보더 글로우 — 카드와 버튼에 퍼플/마젠타 외곽 발광
3. 스캔라인 오버레이 — 배경에 미세한 수평 라인 텍스처

**Interaction Philosophy**: 마우스 호버 시 네온이 점등되는 듯한 반응. 클릭은 전기 방전 같은 즉각적 피드백. 스크롤은 도시의 엘리베이터를 타듯 층을 이동하는 느낌.

**Animation**: 섹션 진입 시 요소들이 글리치하며 나타남(translateX + opacity + clip-path). 네온 라인은 지속적으로 미세하게 펄스. 배경에 파티클이 천천히 부유.

**Typography System**: 제목 — Orbitron (기하학적 미래형, 700-900), 본문 — Space Grotesk (기술적이면서 가독성 높음, 400-500), 강조 — Rajdhani (날카로운 엣지감)

## 아이디어 2: Cosmic Lounge (코스믹 라운지)

**Design Movement**: 스페이스 에이지 럭셔리 + 아르데코의 재해석. 1960년대 우주 시대의 낙관적 미래주의와 아르데코의 기하학적 우아함을 현대적으로 재조합.

**Core Principles**:
1. 우주적 깊이감 — 다층 패럴랙스와 그라디언트로 무한한 공간감 연출
2. 기하학적 리듬 — 반복되는 다이아몬드, 아치, 방사형 패턴으로 시각적 리듬
3. 소재의 대비 — 매트 다크와 광택 메탈릭의 교차로 고급감 극대화
4. 의식적 여백 — 넉넉한 패딩과 마진으로 "호흡하는" 레이아웃

**Color Philosophy**: 딥 네이비(#0F172A)와 미드나이트 블루(#1E293B)가 우주의 깊이를 표현. 골드(#F59E0B)와 앰버(#D97706)가 아르데코적 럭셔리를 상징. 로고의 퍼플(#8B5CF6)이 우주의 신비로움을 연결.

**Layout Paradigm**: 중앙 집중형 타워 레이아웃에서 벗어나, 좌우로 교차하는 지그재그 스크롤 구조. 각 섹션이 우주 정거장의 다른 모듈처럼 독립적 공간감을 가짐.

**Signature Elements**:
1. 별빛 파티클 배경 — 미세한 점들이 천천히 이동하는 우주 배경
2. 아르데코 프레임 — 섹션 경계에 기하학적 장식 라인
3. 홀로그래픽 카드 — 기울기에 따라 색상이 변하는 카드 효과

**Interaction Philosophy**: 호버 시 요소가 부드럽게 떠오르는 무중력 효과. 스크롤은 우주선이 정거장을 순항하듯 부드럽고 관성적. 클릭은 별이 반짝이는 듯한 광학적 피드백.

**Animation**: 패럴랙스 스크롤로 다층 깊이감. 요소 진입 시 페이드 + 스케일 업(0.95→1.0). 골드 라인이 드로잉되듯 그려지는 SVG 애니메이션.

**Typography System**: 제목 — Playfair Display (클래식 세리프, 아르데코 우아함, 700-900), 본문 — DM Sans (현대적 산세리프, 깔끔한 가독성, 400-500), 숫자/데이터 — JetBrains Mono (기술적 정확성)

## 아이디어 3: Neon Noir (네온 느와르)

**Design Movement**: 필름 느와르 + 한국 네온 스트리트 문화의 결합. 1940년대 느와르 영화의 극적인 명암 대비와 서울/홍대/강남의 네온 간판 문화를 디지털로 재현.

**Core Principles**:
1. 극적 명암 — 순수 블랙과 밝은 네온의 하이 콘트라스트로 시선 유도
2. 수직적 리듬 — 한국 도시의 수직 간판처럼 세로 방향 강조 요소 활용
3. 레이어드 글래스 — 유리/아크릴 질감의 반투명 레이어로 깊이감
4. 타이포 중심 — 한글과 영문의 대담한 혼합 타이포그래피가 주인공

**Color Philosophy**: 제트 블랙(#050505)이 밤의 도시를 형성. 로고에서 추출한 일렉트릭 퍼플(#9333EA)과 핫 핑크(#DB2777)가 네온 사인 역할. 아이스 블루(#38BDF8)가 PC 모니터의 차가운 빛을 상징. 웜 화이트(#F5F5F4)가 텍스트의 가독성 보장.

**Layout Paradigm**: 풀스크린 수직 스택 + 사이드 네비게이션 도트. 각 섹션이 하나의 "씬(Scene)"처럼 전체 뷰포트를 차지하며, 섹션 간 전환이 영화의 장면 전환처럼 연출. 콘텐츠는 비대칭 2컬럼(60/40 또는 70/30)으로 배치.

**Signature Elements**:
1. 네온 사인 타이포 — 주요 제목이 실제 네온 사인처럼 발광하고 미세하게 깜빡임
2. 글래스모피즘 카드 — 반투명 유리 질감의 카드에 배경 블러
3. 라이트 트레일 — 스크롤 시 네온 빛줄기가 따라오는 효과

**Interaction Philosophy**: 호버 시 네온 사인이 켜지듯 글로우 확산. 버튼 클릭 시 빛이 퍼지는 리플 효과. 스크롤은 영화 릴이 돌아가듯 부드러운 스냅 스크롤.

**Animation**: 섹션 진입 시 요소들이 네온이 켜지듯 순차적으로 밝아짐(opacity + glow filter). 배경에 미세한 보케(bokeh) 효과가 떠다님. 텍스트는 타이핑 효과로 등장.

**Typography System**: 제목 — Black Han Sans (한글 대담함) + Bebas Neue (영문 컨덴스드, 간판 느낌), 본문 — Pretendard (한글 최적화 산세리프, 400-500), 강조 — Space Mono (코드/기술적 데이터)

</idea>
