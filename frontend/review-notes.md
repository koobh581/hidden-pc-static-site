# 사이트 리뷰 노트

## 확인된 사항
1. Hero 섹션: 로고 블랙 배경이 보임 - mix-blend-mode: lighten 적용되었으나 PNG 배경이 검정이라 블렌딩됨. 좋아보임.
2. About 섹션: 잘 표시됨, 카드 4개 + 통계 수치
3. Facilities 섹션: 게이밍 셋업 이미지 + 스펙 카드 + 와이드 인테리어 이미지 잘 보임
4. Food 섹션: 음식 이미지 잘 보이나, 오른쪽 메뉴 카드가 보이지 않는 것 같음 (빈 공간)
5. 전체적으로 다크 테마 + 네온 글로우 잘 적용됨

## 수정 필요
- Food 섹션 오른쪽 메뉴 카드 확인 필요 (스크롤 시 보이지 않을 수 있음 - 뷰포트 밖일 수 있음)
- 전체적으로 양호한 상태
